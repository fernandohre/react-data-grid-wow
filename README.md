# React Data Grid WoW

React Grid that displays a customizable table with data from World of Warcraft characters (class, item level, etc.)

This is a fork of the [React Data Grid](https://github.com/adazzle/react-data-grid) made by [Adazzle](https://github.com/adazzle/react-data-grid) for a college project (Software Engineering at Universidade Federal de Goi√°s, Brazil).

# Instalation

npm install react-data-grid-wow
